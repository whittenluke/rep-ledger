# Muscle group mapping (colloquial → anatomical)

Used for filter chips in the system exercise library. Colloquial labels are user-facing; each maps to one or more anatomical `primary_muscle` values in the database.

| Colloquial | Anatomical (primary_muscle) |
|------------|-----------------------------|
| Chest      | Pectorals                   |
| Shoulders  | Deltoids                    |
| Triceps    | Triceps                     |
| Biceps     | Biceps                      |
| Back       | Latissimus Dorsi, Rhomboids, Trapezius |
| Legs       | Quadriceps, Hamstrings, Glutes, Calves |
| Core       | Core, Lower Back            |
